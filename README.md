# 🍺🍻 Beer as a Service 

## Description

BaaS is beer as a service using [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Demo

[Demo on Herokuapp](https://beerasaservice.herokuapp.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
